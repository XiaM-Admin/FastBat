## 这是一个快速启动Bat批处理脚本的后台托盘程序

### 程序主要实现
1. 在程序目录下创建一个文件夹，用于存放bat批处理脚本
2. 在托盘菜单中创建"重载"按钮，用于重载bat批处理脚本
3. 在托盘菜单中创建"退出"按钮，用于退出程序
4. 在托盘菜单中创建一个脚本列表，用于显示bat批处理脚本
5. 点击脚本列表中的脚本，执行bat脚本
